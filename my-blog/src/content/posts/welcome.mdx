## 技术栈选择

这个博客使用了以下技术：

```typescript
const techStack = {
  framework: 'Next.js 14',
  language: 'TypeScript',
  styling: 'Tailwind CSS',
  content: 'MDX',
  deployment: 'Vercel'
}
```

选择这个技术栈的原因：

1. **Next.js**：现代React框架，SSG支持，SEO友好
2. **TypeScript**：类型安全，提升代码质量
3. **Tailwind CSS**：实用优先的CSS框架，开发效率高
4. **MDX**：支持在Markdown中使用React组件
