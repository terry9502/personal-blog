你好！欢迎来到我的个人博客。这是我作为软件工程专业学生的第一篇博客文章。

## 为什么要写博客？

就是在今年的10月份左右，我突发奇想想要做一个博客。我发现中国太多无用的信息非常影响我们的判断，CSDN上虽然有很多有用的信息，但是太乱，充斥着大量的广告，非常影响用户体验。还有最重要的一点就是我自己在CSDN上的文章自动变成会员文章而且我自己无法修改，这一点非常恶心，除非把文章删除。于是我就下定决心做一个博客，分享我的心得体会！

现在随着AI技术的发展，越来越多的文章都是使用AI来生成，不仅仅是文章，好多视频内容现在都可以使用AI技术来生成，这使得我们寻找优质内容变得更加困难，这也是我想要写博客的初衷。

探寻技术背后的真相，不仅仅是技术！

## 技术栈选择

这个博客使用了以下技术：

```typescript
const techStack = {
  framework: 'Next.js 14',
  language: 'TypeScript',
  styling: 'Tailwind CSS',
  content: 'MDX',
  deployment: 'Vercel'
}
```

选择这个技术栈的原因：

1. **Next.js**：现代React框架，SSG支持，SEO友好
2. **TypeScript**：类型安全，提升代码质量
3. **Tailwind CSS**：实用优先的CSS框架，开发效率高
4. **MDX**：支持在Markdown中使用React组件
